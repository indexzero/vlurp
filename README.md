## Usage

**CLI**
```
vlurp <user>/<repo>
vlurp <user>/<repo>
vlurp <url>
```

**Textual routing examples**
```
vlurp <user> <repo> to <target>
vlurp <user>/<repo> to <target>
vlurp <url> to <target>
in <target> vlurp <url>
this email I just got ```<copy/paste slop>``` has a <user>/<repo> in it; please vlurp it
vlurp everything in <text>
```
